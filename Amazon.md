# Amzn-Search  
An API by which you can easily get Amazon search results into JSON fromat.

### Usage:  

```
https://amznsearch.vercel.app/api/?query={your-query-here}
```  
Example: https://amznsearch.vercel.app/api/?query=macbook+pro
